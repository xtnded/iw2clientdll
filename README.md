# iw2clientdll
Call of Duty 2 1.4 Client Patch

# Requirements
- At least PC with Windows XP SP3 Update, but recommended is Windows 10.
- Disabled VirtualStore

# Changes/additions
- Windowed mode
- Imgui
- Discord RPC
- bugfixes (ex. Long blackscreen wait)
